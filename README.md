# Factoid Finance Source Code
Source code for TSA Project Factoid Finance

## Features
- Authentication System
- Static Pages
- Portfolio Dashboard (List View with Graphs etc)
- Detail Asset View
- Fundamental Analysis
- Technical Analysis
- Quant Finance
- News Sentiment Analysis
- Scanner

## Technical Stack
- Django
- HTML/CSS/JS
- PostGres & Redis
- Docker
- Kubernetes
- Github Actions CI/CD
- Logging
- Gunicorn, Nginx
- EC2, S3, RDS
